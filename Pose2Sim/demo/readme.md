# Pose2Sim Demo - Erste Schritte

## Voraussetzungen

Stellen Sie sicher, dass Sie Anaconda und Pose2Sim auf Ihrem System installiert haben.

## Ausführung des Pose2Sim CLI-Skripts

### Schritt 1: Öffnen Sie das Anaconda-Terminal

Suchen Sie im Startmenü nach "Anaconda Prompt" und öffnen Sie es

### Schritt 2: Navigieren Sie zum Projekt-Repository

Öffnen Sie Ihr Anaconda-Terminal und navigieren Sie zum Demo-Ordner:

```bash
cd path\to\demo
```

Ersetzen Sie `path\to` durch den tatsächlichen Pfad zu Ihrem Projektverzeichnis.

### Schritt 3: Aktivieren Sie die Conda-Umgebung

Aktivieren Sie die pose2sim-Umgebung:

```bash
conda activate pose2sim
```

### Schritt 4: Führen Sie das Skript aus

Führen Sie das Pose2Sim CLI-Skript aus:

```bash
python pose2sim_cli.py
```

Das Skript verarbeitet Ihre Eingabedaten gemäß der Konfiguration in `Config.toml`.

# Pose2Sim Demo - Getting Started

## Prerequisites

Make sure you have Anaconda and Pose2Sim installed on your system.

## Running the Pose2Sim CLI Script

### Step 1: Open Anaconda Terminal

Search for "Anaconda Prompt" in your Start Menu and open it

### Step 2: Navigate to the Project Repository

Open your Anaconda terminal and navigate to the demo folder:

```bash
cd path\to\demo
```

Replace `path\to` with the actual path to your project directory.

### Step 3: Activate the Conda Environment

Activate the pose2sim environment:

```bash
conda activate pose2sim
```

### Step 4: Run the Script

Execute the Pose2Sim CLI script:

```bash
python pose2sim_cli.py
```

The script will process your input data according to the configuration in `Config.toml`.
